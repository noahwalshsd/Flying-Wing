# Flying-Wing
A flying wing to be demod at Outpost. I really want to go to outpost!!!
This is a flying wing plane that will eventually have self-leveling capabilities due to the onboard accelerometer. It has a Rudder sticking out the top and middle to increase horizontal stability without increasing tail-heavyness. My biggest Problem was dealing with the weight of the tail. I have neber made an rc plane before, so a flying wing for my first time will be very unique. My cat helped me code. AI was not used in this project besides as a basic suggester on some parts and to double-check my code. 
The Price of the battery just went up, so it is a little more expensive than when I submitted
